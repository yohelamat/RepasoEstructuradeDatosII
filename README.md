# Repaso Estructura de Datos II 📦

Este repositorio contiene el código fuente de mi proyecto integrador para la materia de Estructura de Datos II. El programa (`repaso_estructura.py`) simula un sistema de inventario que utiliza cinco estructuras de datos diferentes para optimizar distintas tareas del negocio:

* **Arreglos:** Para la carga estática inicial y ordenamiento por precio.
* **Listas Enlazadas:** Para el dinamismo de agregar o eliminar productos.
* **Pilas (LIFO):** Para llevar el historial de acciones y la función "Deshacer".
* **Colas (FIFO):** Para simular la atención de una fila de clientes.
* **Árbol Binario de Búsqueda (ABB):** Para organizar los códigos jerárquicamente e imprimir el inventario ordenado (recorrido Inorden).

### 📄 Nota Importante sobre la Evaluación
Toda la sustentación técnica, la prueba de escritorio, las evidencias de ejecución y la justificación de por qué se usó cada estructura de datos se encuentran detalladas en el **documento PDF adjunto** en la entrega oficial de la plataforma universitaria.
